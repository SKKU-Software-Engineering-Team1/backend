# SKKUNION - Backend

---

## 소프트웨어공학개론 Team 1

## 📒 프로젝트 소개
SpringBoot를 사용한 동아리 지원 밎 부원 모집 플랫폼(웹서비스) SKKUNION의 Backend

## 👫 백엔드 팀 멤버
최은서, 백명규, 정윤석

## ⏳ 개발 기간
2023.09 - 2023.11

## 🔩 개발환경
- JAVA 11
- SpringBoot 2.7.16
- MySQL 8.0.34
- ORM : JPA

## 🪽주요 기능
- 회원 관련
    1. 로그인(토큰 및 리프레시 토큰 관리)
    2. 회원가입
    3. 회원정보수정
- 동아리 지원 관련
    1. 동아리 정보 리스트 조회
    2. 동아리 정보 단건조회
- 동아리 부원 모집 관련
    1. 사용자 리스트 조회
    2. 사용자 단건 조회
    3. 동아리 정보 수정
- 채팅 관련
    1. 채팅 전송
    2. 채팅방 생성
    3. 채팅방 정보 조회
